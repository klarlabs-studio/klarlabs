# Klarlabs

> We build tools that make complex things clear.

Precision software from Munich. This monorepo holds the Klarlabs design system and studio sites.

## Packages

| Package | Description |
| --- | --- |
| [`@klarlabs-studio/ui`](packages/ui) | Design system — Lit web components, design tokens, D3 charts |

## Development

```bash
pnpm install
pnpm dev      # component demo (packages/ui)
pnpm test     # all tests
pnpm build    # all packages
```
